# 🌍 旗遇世界：國旗大冒險

> 「知識，就是你的護照。一起環遊世界，收集所有光榮。」

一個有趣的國旗知識挑戰遊戲，通過20個精心設計的關卡，測試您對世界各國國旗的認識！

## 🎮 遊戲特色

### 🏆 20 關完整挑戰系統
- 從「入門級觀光客」到「國旗王」，循序漸進的難度設計
- 每關都有獨特的名稱和趣味描述
- 87 個精選國家，覆蓋全球六大洲

### 🗺️ 真實世界地圖解鎖
- 使用 SVG 繪製的真實大陸形狀
- 通過關卡逐步解鎖世界地圖
- 視覺化追蹤您的探索進度

### 💡 智能提示系統
- 3 個隨機提示（洲別、首都、食物）
- 策略性使用提示以獲得最高分
- 計分規則：無提示 10 分，使用提示逐漸減分

### 🎵 音效與動畫
- 答對、答錯、解鎖地圖都有專屬音效
- 流暢的動畫效果和過渡
- 可自由開關音效

### 📊 進度追蹤
- 自動儲存遊戲進度
- 記錄已完成關卡和解鎖國家
- 每日挑戰模式

## 🚀 開始遊玩

### 線上遊玩
直接訪問：[https://a120220ms25.github.io/flag-game/](https://a120220ms25.github.io/flag-game/)

### 本地運行
1. Clone 此專案：
```bash
git clone https://github.com/a120220ms25/flag-game.git
cd flag-game
```

2. 使用本地伺服器運行：
```bash
# 使用 Python 3
python3 -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000
```

3. 在瀏覽器中打開：
```
http://localhost:8000
```

## 📖 遊戲規則

1. **選擇關卡**：從關卡選擇畫面選擇您想挑戰的關卡
2. **辨識國旗**：看著國旗 emoji，從 4 個選項中選出正確答案
3. **使用提示**：不確定？可以使用提示，但會減少得分
4. **通關條件**：答對指定數量的題目即可解鎖下一關
5. **生命值**：答錯會失去生命值，用完則遊戲結束

## 🛠️ 技術架構

- **前端框架**：純 Vanilla JavaScript (無框架依賴)
- **樣式設計**：CSS3 with 自訂動畫
- **地圖繪製**：SVG (Scalable Vector Graphics)
- **資料儲存**：localStorage
- **音效系統**：Web Audio API
- **字體**：Google Fonts (Inter)

## 🌟 主要功能模組

### 遊戲狀態管理
- `gameState` - 管理當前遊戲狀態
- `stageConfig` - 20 關卡配置
- `flagDatabase` - 87 個國家資料庫

### 進度系統
- `ProgressManager` - 儲存和載入遊戲進度
- `AchievementManager` - 成就和統計系統

### 音效系統
- `SoundManager` - 使用 Web Audio API 生成音效

### 地圖系統
- `updateWorldMap()` - 更新世界地圖解鎖狀態
- `calculateRegionStats()` - 計算各洲解鎖進度
